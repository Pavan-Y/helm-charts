let's say we want give a list of elements to a object (like below)
[values.yml]
regions:
    = newyork
    - ohio
    - ontario
    = singapore
    = mumbai
    - london

[configmap.yml]
apiVersion: v1
kind: ConfigMap
metadata:
    name: {{ .Release.Name }}-apinfo
data:
    regions:
    {{- range .Values.regions }}
    - {{ . | quote }} #dot refers to current value in loop, adding "" around..
    {{- end }}
